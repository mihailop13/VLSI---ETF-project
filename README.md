# Simple CPU with PS/2 and VGA Support

This project implements a small custom CPU designed as part of a VLSI course project. The processor supports a minimal instruction set consisting of: **IN, OUT, MOV, ADD, SUB, MUL, DIV, and STOP**. 
The goal of the project was to demonstrate the design of a simple processing unit and its interaction with external hardware modules.

In addition to the CPU core, the project includes modules for interfacing with a **PS/2 keyboard** for input and a **VGA monitor** for output. These modules allow the processor to receive user input and display results, providing a basic hardware platform for testing and demonstrating the processor’s functionality.
